# msl_watcher - хром пашырэнне для аматараў мафii.

msl_watcher дапаможа глядзець мафiю з ляўшой без ралей ***без падпiскi на бусцi***.

## тры рэжымы:

1. нацiснi alt+1 (macos ⸺ opt+1) калi за сталом дзень ⸺ ролi на гульню, праверкi i адстрэлы будуць схаваныя. !(/assets/day_mode_example.png)
2. нацiснi alt+2 (macos ⸺ opt+1) калi за сталом ноч ⸺ усё, што адбываецца за сталом ноччу, будзе схавана. гук будзе адключаны. !(/assets/night_mode_example.png)
2. нацiснi alt+9 (macos ⸺ opt+9) каб адключыць хаванне элементаў.
