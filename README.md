# msl_watcher — хром пашырэнне для аматараў мафii.

msl_watcher дапаможа глядзець мафiю з ляўшой без роляў **без падпiскi на бусцi**.

## тры рэжымы:

1. нацiснi **alt+1** (macos — **opt+1**) калi за сталом дзень — ролi на гульню, праверкi i адстрэлы будуць схаваныя. <br><br>![прыклад](/assets/day_mode_example.png)
2. нацiснi **alt+2** (macos — **opt+2**) калi за сталом ноч — усё, што адбываецца за сталом ноччу, будзе схавана. гук будзе адключаны аўтаматычна. <br><br>![прыклад](/assets/night_mode_example.png)
3. нацiснi **alt+9** (macos — **opt+9**) каб адключыць хаванне элементаў.

---

## як усталяваць:

каб пашырэнне запрацавала ў вашым браўзеры, выканайце наступныя крокі:

1. спампуй [тэчку з кодам](https://github.com/matraass11/msl_watcher/raw/main/msl_watcher.zip?download=) i раcпакуй.
2. адкрый google chrome і перайдзі па адрасе `chrome://extensions/`.
3. у правым верхнім куце ўключы **"рэжым распрацоўніка"** (developer mode).
4. націсні на кнопку **"загрузіць распакаванае пашырэнне"** (load unpacked) злева зверху.
5. абяры тэчку `msl_watcher` у дыялогавым акне.
